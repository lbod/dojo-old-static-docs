dijit._editor.plugins.EnterKeyHandling (version 1.3)

Object » dijit._editor._Plugin » dijit._editor.plugins.EnterKeyHandling

Usage:

var foo = new dijit._editor.plugins.EnterKeyHandling();

This plugin tries to make all browsers have identical behavior when the user presses the ENTER key. Specifically, it fixes the double-spaced line problem on IE.
