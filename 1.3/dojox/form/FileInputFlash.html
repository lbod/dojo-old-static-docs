fileMask
Defined by dojox.form.FileInputFlash

Array[Description, FileTypes], Array[...]...] */ (an array, or an array of arrays) Restrict file selection to certain file types Empty array defaults to "All Files" Usage:

fileMask = ["Images", ".jpg;.jpeg;.gif;.png"] or fileMask = [

["Jpeg File", ".jpg;.jpeg"], ["GIF File", ".gif"], ["PNG File", ".png"], ["All Images", ".jpg;.jpeg;.gif;.png"], ] NOTE: MacType is not supported, as it does not work very well. fileMask will work on a Mac, but differently than Windows. The second example above in Windows will mask All but the selected file type, shown in a drop-down at the bottom of the system dialog. In Mac, all types in all arrays will be shown and non-types masked.
