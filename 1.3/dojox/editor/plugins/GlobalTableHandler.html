dojox.editor.plugins.GlobalTableHandler (version 1.3)

Object » dijit._editor._Plugin » dojox.editor.plugins.GlobalTableHandler

Usage:

var foo = new dojox.editor.plugins.GlobalTableHandler();

A global object that handles common tasks for all the plugins. Since there are several plugins that are all calling common methods, it's preferable that they call a centralized location that either has a set variable or a timeout to only repeat code-heavy calls when necessary.
