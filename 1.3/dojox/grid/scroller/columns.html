Object » dojox.grid.scroller.base » dojox.grid.scroller » dojox.grid.scroller.columns

Usage:

var foo = new dojox.grid.scroller.columns();

Virtual scroller class that scrolls list of columns. Owned by grid and used internally for virtual scrolling.
