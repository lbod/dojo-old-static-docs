The keyword arguments object that is used to create a 5 color palette based on the analogous rules as implemented at http://kuler.adobe.com, using the HSV color model.

Property Summary

  • base The base color to be used to generate the palette.
  • high The difference between the hue of the base color and the highest hue. In degrees, default is 60.
  • low The difference between the hue of the base color and the lowest hue. In degrees, default is 18.

Method Summary

  • constructor(base, high, low) returns dojox.color.Palette.__analogousArgs The base color to be used to generate the palette.
