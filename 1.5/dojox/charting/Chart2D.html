Usage:

var foo = new dojox.charting.Chart2D(/* DOMNode */ node, /* dojox.charting.__Chart2DCtorArgs? */ kwArgs);

dojox.charting.Chart2D is the primary object used for any kind of charts. It is simple to create--just pass it a node reference, which is used as the container for the chart--and a set of optional keyword arguments and go. Note that like most of dojox.gfx, most of dojox.charting.Chart2D's methods are designed to return a reference to the chart itself, to allow for functional chaining. This makes defining everything on a Chart very easy to do.

Examples:

Example 1

Create an area chart, with smoothing. 
	new dojox.charting.Chart2D(node))
		.addPlot("default", { type: "Areas", tension: "X" })
		.setTheme(dojox.charting.themes.Shrooms)
		.addSeries("Series A", [1, 2, 0.5, 1.5, 1, 2.8, 0.4])
		.addSeries("Series B", [2.6, 1.8, 2, 1, 1.4, 0.7, 2])
		.addSeries("Series C", [6.3, 1.8, 3, 0.5, 4.4, 2.7, 2])
		.render();

Example 2

The form of data in a data series can take a number of forms: a simple array, an array of objects {x,y}, or something custom (as determined by the plot). Here's an example of a Candlestick chart, which expects an object of { open, high, low, close }.
