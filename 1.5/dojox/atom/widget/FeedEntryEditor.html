Object » dijit._Widget » dojox.atom.widget.FeedEntryViewer » dojox.atom.widget.FeedEntryEditor
dojo.require("dojox.atom.widget.FeedEntryEditor");
Defined in dojox/atom/widget/FeedEntryEditor.js

Usage:

var foo = new dojox.atom.widget.FeedEntryEditor(/* Object? */ params, /* DomNode|String */ srcNodeRef);

An ATOM feed entry editor that allows viewing of the individual attributes of an entry. Flag denoting if the current entry is editable or not.

	var button = new dijit.form.Button({ label:"click" }).placeAt("wrapper","first");
