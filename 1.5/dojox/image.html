dojox.image.preload
Defined by dojox.image
Array An Array of DomNodes that have been cached.
Preload a list of images in the dom.
ParameterTypeDescription
urlsArrayThe list of urls to load. Can be any valid .src attribute.
Example 1
Load two images into cache: 
	dojox.image.preload(["foo.png", "bar.gif"]);
Example 2
Using djConfig: 
	var djConfig = {
		preloadImages:["bar.png", "baz.png", "http://example.com/icon.gif"]
	};
