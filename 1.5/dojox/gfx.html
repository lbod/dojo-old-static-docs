dojox.gfx (version 1.5)

Object » dojox.gfx

dojox.gfx is an advanced API providing normalized vector drawing in a variety of browsers. It has individual renderers for SVG, VML, Canvas, and Silverlight.

dojox.gfx.decompose
Defined by dojox.gfx
Returns Object

this function decompose a matrix into four logical components: translation, rotation, scaling, and one more rotation using SVD. The components should be applied in following order: [translate, rotate(angle2), scale, rotate(angle1)]

dojox.gfx.splitFontString
Defined by dojox.gfx
Returns Object

Converts a CSS font string to a gfx font object. The CSS font string components should follow the W3C specified order (see http://www.w3.org/TR/CSS2/fonts.html#font-shorthand): style, variant, weight, size, optional line height (will be ignored), and family.

ParameterTypeDescription
