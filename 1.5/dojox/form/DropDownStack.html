  • _onDropDownMouseUp(e) Callback when the user lifts their mouse after mouse down on the arrow icon. If the drop is a simple menu and the mouse is over the menu, we execute it, otherwise, we focus our dropDown node. If the event is missing, then we are not a mouseup event. This is useful for the common mouse movement pattern with native browser <select> nodes: 1. mouse down on the select node (probably on the arrow) 2. move mouse to a menu item while holding down the mouse button 3. mouse up. this selects the menu item as though the user had clicked it.
