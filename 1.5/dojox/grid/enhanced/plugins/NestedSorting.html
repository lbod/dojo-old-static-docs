Property Summary

  • _minColWidth Used for calculating min cell width, will be updated dynamically 58
  • _minColWidthUpdated Flag to indicate whether the min col width has been updated
  • _overResizeWidth Overwrite the default over resize width, so that the resize cursor is more obvious when leveraged with sorting hover tips
  • _sortTipMap Cache the tip on/off status for each cell, e.g. {cellIndex: true|false}
  • _unarySortCell Cache for the current unary sort cell(the 1st column in sorting sequence) will be set as {cell: cell, cellNode: node}
  • _widthDelta Min width delta 18
  • exceptionalSelectedItems Cache data store items with exceptional selection state. Used to retain selection states accross sortings. User may first select/deselect all rows and then deselect/select certain rows, these later changed rows have a different state with the global selection state, that is exceptional selection state
  • inSorting
  • sortAttrs Sorting attributes, e.g.[{attr: 'col1', asc: 1|-1|0, cell: cell, cellNode: node}, {...}, ...]
  • storeItemSelected Attribute used in data store to mark which row(s) are selected accross sortings
  • toggleAllValue

  • _selectionChanged(obj) Subscriber of rowSelectionChangedTopic, update global row selection state accordingly

  • _toggleHighlight(cellView, e, allOff) Toggle hover state for selection region, unary sort region (unarySortWrapper) and nested sort region (unarySortWrapper)
  • _toggleProgressTip(on, e) Change the cursor to progress or vice versa
  • _unarySortInFocus() See if the unary sort node is in keyboard focus return: Boolean

  • sortStateInt2Str(si) returns 'SortUp';|'SortDown'; Sorting state integer return: String Sort info string
