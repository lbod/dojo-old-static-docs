dojox.grid.__DataCellDef (version 1.5)

Object » dojox.grid.__CellDef » dojox.grid.__DataCellDef
dojo.require("dojox.grid.DataGrid");
Defined in dojox/grid/DataGrid.js

Usage:

var foo = new dojox.grid.__DataCellDef();

Property Summary

  • attrs A space separated string of attribute='value' pairs to add to the header cell element and main grid cell elements.
  • cellClasses A space separated list of classes to apply to just the main grid cells.
  • cellStyles A string of styles to apply to just the main grid cells. Must end in a ';'
  • classes A space separated list of classes to apply to both the header cell and the main grid cells.
  • colSpan How many columns to span this cell. Will not work in the first sub-row of cells.
  • editable Whether this cell should be editable or not.
  • headerClasses A space separated list of classes to apply to just the header cell.
  • headerStyles A string of styles to apply to just the header cell. Must end in a ';'
  • hidden If true, the cell will not be displayed. noresize: Boolean? If true, the cell will not be able to be resized.
  • name The text to use in the header of the grid for this cell.
  • rowSpan How many sub-rows to span this cell.
  • styles A string of styles to apply to both the header cell and main grid cells. Must end in a ';'.
  • type TODO
  • value If "get" is not specified, this is used as the data for the cell. defaultValue: String? If "get" and "value" aren't specified or if "get" returns an undefined value, this is used as the data for the cell. "formatter" is not run on this if "get" returns an undefined value.
  • width A CSS size. If it's an Integer, the width will be in em's.

Method Summary

  • constructor() returns dojox.grid.__DataCellDef A reference to the constructor function for this object.
  • formatter() function(data, rowIndex){} data is of type anything, rowIndex is of type Integer. This function will be called after the cell has its data but before it passes it back to the grid to render. Returns the formatted version of the cell's data.
  • get() function(rowIndex){} rowIndex is of type Integer. This function will be called when a cell requests data. Returns the unformatted data for the cell.
