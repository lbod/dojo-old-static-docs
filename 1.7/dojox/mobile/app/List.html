dojox.mobile.app.List (version 1.7)

Object » dojo.Stateful » dijit._WidgetBase » dojox.mobile.app.List

Usage:

var foo = new dojox.mobile.app.List(/* Object? */ params, /* DomNode|String */ srcNodeRef);

A templated list widget. Given a simple array of data objects and a HTML template, it renders a list of elements, with support for a swipe delete action. An optional template can be provided for when the list is empty.
