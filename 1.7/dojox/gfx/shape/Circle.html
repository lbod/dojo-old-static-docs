Method Summary

  • _applyTransform() returns self physically sets a matrix
  • _getRealMatrix() returns dojox.gfx.Matrix2D returns the cumulative ("real") transformation matrix by combining the shape's matrix with its parent's matrix
  • _moveToBack() renderer-specific hook, see dojox.gfx.shape.Shape.moveToFront()
  • _moveToFront() renderer-specific hook, see dojox.gfx.shape.Shape.moveToFront()
  • _setParent(parent, matrix) returns self sets a parent
  • _updateParentMatrix(matrix) returns self updates the parent matrix with new matrix
  • applyLeftTransform(matrix) returns self multiplies the existing matrix with an argument on left side (matrix * this.matrix)
  • applyRightTransform(matrix) returns self multiplies the existing matrix with an argument on right side (this.matrix * matrix)
  • applyTransform(matrix) returns self a shortcut for dojox.gfx.Shape.applyRightTransform
  • constructor(rawNode) returns dojox.gfx.shape.Circle a DOM Node
  • getBoundingBox() returns dojox.gfx.Rectangle returns the bounding box
  • getEventSource() returns Node returns a Node, which is used as a source of events for this shape
  • getFill() returns Object returns the current fill object or null (see dojox.gfx.defaultLinearGradient, dojox.gfx.defaultRadialGradient, dojox.gfx.defaultPattern, or dojo.Color)
  • getNode() returns Node returns the current DOM Node or null
  • getParent() returns Object returns the parent or null (see dojox.gfx.Surface, dojox.gfx.shape.VirtualGroup, or dojox.gfx.Group)
  • getShape() returns Object returns the current shape object or null (see dojox.gfx.defaultPath, dojox.gfx.defaultPolyline, dojox.gfx.defaultRect, dojox.gfx.defaultEllipse, dojox.gfx.defaultCircle, dojox.gfx.defaultLine, or dojox.gfx.defaultImage)
  • getStroke() returns Object returns the current stroke object or null (see dojox.gfx.defaultStroke)
  • getTransform() returns dojox.gfx.Matrix2D returns the current transformation matrix or null
  • getTransformedBoundingBox() returns null|Array returns an array of four points or null four points represent four corners of the untransformed bounding box
  • moveToBack() moves a shape to back of its parent's list of shapes
  • moveToFront() returns self moves a shape to front of its parent's list of shapes
  • removeShape(silently) returns self removes the shape from its parent's list of shapes
  • setFill(fill) returns self sets a fill object (the default implementation simply ignores it)
  • setShape(shape) returns self sets a shape object (the default implementation simply ignores it)
  • setStroke(stroke) returns self sets a stroke object (the default implementation simply ignores it)
  • setTransform(matrix) returns self sets a transformation matrix
