dojox.date.timezone (version 1.7)

Object » dojox.date.timezone
dojo.require("dojox.date.timezone");
Defined in dojox/date/timezone.js

mix-in to dojo.date to provide timezones based on the Olson timezone data. If you pass "timezone" as a parameter to your format options, then you get the date formatted (and offset) for that timezone

Method Summary

  • dojox.date.timezone.getAllZones() Returns an array of zones that have been loaded
  • dojox.date.timezone.getTzInfo(dt, tz) Returns the timezone information for the given date and timezone string
  • dojox.date.timezone.loadZoneData(data) Loads the given data object into the zone database
