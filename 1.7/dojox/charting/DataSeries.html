  • value Function, which takes a store, and an object handle, and produces an output possibly inspecting the store's item. Or a dictionary object, which tells what names to extract from an object and how to map them to an output. Or a string, which is a numeric field name to use for plotting. If undefined, null or empty string (the default), "value" field is extracted.
