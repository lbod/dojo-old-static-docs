dojo.require("dojox.data.PicasaStore");
Defined in dojox/data/PicasaStore.js

Usage:

var foo = new dojox.data.PicasaStore(/* Object */ args);

The PicasaStore is a Datastore interface to one of the basic services of the Picasa service, the public photo feed. This does not provide access to all the services of Picasa.

This store cannot do * and ? filtering as the picasa service provides no interface for wildcards.
