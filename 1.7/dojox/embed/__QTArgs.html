Property Summary

  • height The height of the embedded movie; the default value is 240px
  • id A unique key that will be used as the id of the created markup. If you don't provide this, a unique key will be generated.
  • params A set of key/value pairs that you want to define in the resultant markup.
  • path The URL of the movie to embed.
  • redirect A url to redirect the browser to if the current QuickTime version is not supported.
  • width The width of the embedded movie; the default value is 320px.
