dojox.flash.Embed (version 1.7)

Object » dojox.flash.Embed
dojo.require("dojox.flash._base");
Defined in dojox/flash/_base.js

Usage:

var foo = new dojox.flash.Embed(/* */ visible);

Writes out the necessary tags to embed a Flash file into the page. Note that these tags are written out as the page is loaded using document.write, so you must call this class before the page has finished loading.

Property Summary

  • _visible
  • height The height of this Flash applet. The default is the minimal height necessary to show the Flash settings dialog. Current value is 138 pixels.
  • id The id of the Flash object. Current value is 'flashObject'.
  • width The width of this Flash applet. The default is the minimal width necessary to show the Flash settings dialog. Current value is 215 pixels.
