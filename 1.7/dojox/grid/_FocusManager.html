dojox.grid._FocusManager (version 1.7)

Object » dojox.grid._FocusManager
dojo.require("dojox.grid._FocusManager");
Defined in dojox/grid/_FocusManager.js

Usage:

var foo = new dojox.grid._FocusManager(/* */ inGrid);

Controls grid cell focus. Owned by grid and used internally for focusing. Note: grid cell actually receives keyboard input only when cell is being edited.
