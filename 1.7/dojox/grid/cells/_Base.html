dojox.grid.cells._Base (version 1.7)

Object » dojox.grid.cells._Base
dojo.require("dojox.grid.cells._base");
Defined in dojox/grid/cells/_base.js

Usage:

var foo = new dojox.grid.cells._Base(/* */ inProps);

Respresents a grid cell and contains information about column options and methods for retrieving cell related information. Each column in a grid layout has a cell object and most events and many methods provide access to these objects. private
