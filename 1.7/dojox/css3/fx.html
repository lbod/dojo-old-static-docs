Method Summary

  • dojox.css3.fx.bounce(args) Returns an animation that do a "bounce" effect on args.node
  • dojox.css3.fx.expand(args) Returns an animation that expands args.node
  • dojox.css3.fx.flip(args) Returns an animation that flips an element around his y axis
  • dojox.css3.fx.puff(args) Returns an animation that will do a "puff" effect on the given node
  • dojox.css3.fx.rotate(args) Returns an animation that rotates an element
  • dojox.css3.fx.shrink(args) Returns an animation that shrinks args.node
