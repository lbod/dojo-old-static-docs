dijit/layout/StackContainer.ChildWidgetProperties (version 1.8)

These properties can be specified for the children of a StackContainer.

See the dijit/layout/StackContainer.ChildWidgetProperties reference documentation for more information.

Property Summary

  • closable True if user can close (destroy) this child, such as (for example) clicking the X on the tab.
  • disabled Specifies that the button to select this pane should be disabled.
  • iconClass CSS Class specifying icon to use in label associated with this pane.
  • selected Specifies that this widget should be the initially displayed pane.
  • showTitle When true, display title of this widget as tab label etc., rather than just using icon specified in iconClass

Properties

closable
Defined by: dijit/layout/StackContainer

True if user can close (destroy) this child, such as (for example) clicking the X on the tab.

disabled
Defined by: dijit/layout/StackContainer

Specifies that the button to select this pane should be disabled. Doesn't affect programmatic selection of the pane, nor does it deselect the pane if it is currently selected.

iconClass
Defined by: dijit/layout/StackContainer

CSS Class specifying icon to use in label associated with this pane.

selected
Defined by: dijit/layout/StackContainer

Specifies that this widget should be the initially displayed pane. Note: to change the selected child use dijit/layout/StackContainer.selectChild

showTitle
Defined by: dijit/layout/StackContainer

When true, display title of this widget as tab label etc., rather than just using icon specified in iconClass
