dijit/form/_CheckBoxMixin (version 1.8)

Mixin to provide widget functionality corresponding to an HTML checkbox

User interacts with real html inputs. On onclick (which occurs by mouse click, space-bar, or using the arrow keys to switch the selected radio button), we update the state of the checkbox/radio.

See the dijit/form/_CheckBoxMixin reference documentation for more information.

  • _aria_attr
  • _setLabelAttr
  • readOnly Should this widget respond to user input?
  • type type attribute on <input> node.
  • value As an initialization parameter, equivalent to value field on normal checkbox (if checked, the value is passed as the value when form is submitted).
