dojox/gfx3d/scheduler.BinarySearchTree (version 1.8)

Usage:

var foo = new dojox/gfx3d/scheduler.BinarySearchTree(obj,outline);
Defined by dojox/gfx3d/scheduler

build the binary search tree, using binary space partition algorithm. The idea is for any polygon, for example, (a, b, c), the space is divided by the plane into two space: plus and minus.

for any arbitrary vertex p, if(p - a) dotProduct n = 0, p is inside the plane,

0, p is in the plus space, vice versa for minus space. n is the normal vector that is perpendicular the plate, defined as:

    n = ( b - a) crossProduct ( c - a )

in this implementation, n is declared as normal, ,a is declared as orient.
