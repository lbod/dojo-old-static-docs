dojox/layout/TableContainer.ChildWidgetProperties (version 1.8)

Properties to be set on children of TableContainer

See the dojox/layout/TableContainer.ChildWidgetProperties reference documentation for more information.

Property Summary

  • colspan The number of columns this widget should span.
  • label The label to display for a given widget
  • spanLabel Setting spanLabel to true makes the widget take up both the label and value cells.
  • title The label to display for a given widget.

Defined by: dojox/layout/TableContainer

The label to display for a given widget. This is interchangeable with the 'label' parameter, as some widgets already have a use for the 'label', and this can be used instead to avoid conflicts.
