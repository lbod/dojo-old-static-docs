dojox/form/uploader/plugins/HTML5 (version 1.8)

A plugin for dojox.form.Uploader that adds HTML5 multiple-file upload capabilities and progress events.

Add this plugin to have HTML5 capabilities in the Uploader. Note that it does not add these capabilities to browsers that don't support them. For IE or older browsers, add additional plugins: IFrame or Flash.

addDropTarget(node, onlyConnectDrop)
Defined by dojox/form/uploader/plugins/HTML5

Add a dom node which will act as the drop target area so user can drop files to this node.

If onlyConnectDrop is true, dragenter/dragover/dragleave events won't be connected to dojo.stopEvent, and they need to be canceled by user code to allow DnD files to happen. This API is only available in HTML5 plugin (only HTML5 allows DnD files).
