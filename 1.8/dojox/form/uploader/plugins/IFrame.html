dojox/form/uploader/plugins/IFrame (version 1.8)

A plugin for dojox.form.Uploader that adds Ajax upload capabilities.

Only supported by IE, due to the specific iFrame hack used. The formUploaderPluginsHTML5 plugin should be used along with this to add HTML5 capabilities to browsers that support them. Progress events are not supported. Inherits all properties from dojox.form.Uploader and formUploaderPluginsHTML5.
