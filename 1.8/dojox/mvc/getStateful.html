dojox/mvc/getStateful (version 1.8)

Create a dojo/Stateful object from a raw value.

Recursively iterates the raw value given, and convert them to stateful ones.

Parameter Type Description
value Anything

The raw value.
options dojox/mvc/getStatefulOptions

The object that defines how model object should be created from plain object hierarchy.

See the dojox/mvc/getStateful reference documentation for more information.

Method Summary

  • constructor(value, options) Create a dojo/Stateful object from a raw value.
  • getStatefulArray(a) Returns the stateful version of the given array.
  • getStatefulObject(o) Returns the stateful version of the given object.
  • getStatefulValue(v) Just returns the given value.
  • getType(v) Returns the type of the given value.
