dojox/css3/transit (version 1.8)

Performs a transition to hide a node and show another node.

This module defines the transit method which is used to transit the specific region of an application from one view/page to another view/page. This module relies on utilities provided by dojox/css3/transition for the transition effects.

Parameter Type Description
from DomNode
to DomNode
options Object
Optional

The argument to specify the transit effect and direction. The effect can be specified in options.transition. The valid values are 'slide', 'flip', 'fade', 'none'. The direction can be specified in options.reverse. If it is true, the transit effects will be conducted in the reverse direction to the default direction.
