dojox/html/styles._ContentSetter (version 1.8)

Usage:

var foo = new dojox/html/styles._ContentSetter(params,node);
Defined by dojo/html

Provides a configurable, extensible object to wrap the setting on content on a node call the set() method to actually set the content..

Parameter Type Description
params Object
node String | DomNode

See the dojox/html/styles._ContentSetter reference documentation for more information.
