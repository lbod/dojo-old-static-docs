dojox/mobile/TransitionEvent (version 1.8)

A class used to trigger view transitions.

Usage:

var foo = new dojox/mobile/TransitionEvent(target,transitionOptions,triggerEvent);
Defined by dojox/mobile/TransitionEvent

Creates a transition event.

Parameter Type Description
target DomNode

The DOM node that initiates the transition (for example a ListItem).
transitionOptions Object

Contains the transition options.
triggerEvent Event
Optional

The event that triggered the transition (for example a touch event on a ListItem).

See the dojox/mobile/TransitionEvent reference documentation for more information.

Method Summary

  • constructor(target, transitionOptions, triggerEvent) Creates a transition event.
  • dispatch() Dispatches this transition event.
  • endTransition(results) Called when the transition ends.
