dojox/mobile/DatePicker (version 1.8)

A wrapper widget around SpinWheelDatePicker or ValuePickerDatePicker. Returns ValuePickerDatePicker when the current theme is "android". Returns SpinWheelDatePicker otherwise.

See the dojox/mobile/DatePicker reference documentation for more information.
