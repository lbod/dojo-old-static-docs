dojox/mobile/Badge (version 1.8)

A utility to create/update a badge node.

Badge is not a widget, but just a convenience class. A badge consists of a simple DOM button.

Usage:

var foo = new dojox/mobile/Badge(params,node);
Defined by dojox/mobile/Badge

Creates a new instance of the class.

Parameter Type Description
params Object
Optional

Contains properties to be set.
node DomNode
Optional

The DOM node. If none is specified, it is automatically created.

See the dojox/mobile/Badge reference documentation for more information.

Property Summary

  • className A CSS class name of a DOM button.
  • fontSize Font size in pixel.
  • value A text to show in a badge.

Method Summary

  • constructor(params, node) Creates a new instance of the class.
  • getValue() Returns the text shown in the badge.
  • setValue(value) Set a label text to the badge.
