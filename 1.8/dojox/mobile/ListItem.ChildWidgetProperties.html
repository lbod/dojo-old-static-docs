dojox/mobile/ListItem.ChildWidgetProperties (version 1.8)

These properties can be specified for the children of a dojox/mobile/ListItem.

See the dojox/mobile/ListItem.ChildWidgetProperties reference documentation for more information.

Property Summary

  • layout Specifies the position of the ListItem child ("left", "center" or "right").
  • preventTouch Disables touch events on the ListItem child.
