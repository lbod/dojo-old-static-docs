dojox/mobile/i18n.I18NProperties (version 1.8)

These properties can be specified for any widget once the dojox/mobile/i18n module is loaded.

See the dojox/mobile/i18n.I18NProperties reference documentation for more information.

Property Summary

  • mblNoConv Disables localization by dojox/mobile/i18n for the widget on which the property is set.
