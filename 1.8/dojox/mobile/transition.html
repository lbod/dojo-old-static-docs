dojox/mobile/transition (version 1.8)

This is the wrapper module which loads dojox/css3/transit conditionally. If mblCSS3Transition is set to 'dojox/css3/transit', it will be loaded as the module to conduct view transitions, otherwise this module returns null.

See the dojox/mobile/transition reference documentation for more information.
