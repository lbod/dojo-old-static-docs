dojox/mobile/_ContentPaneMixin (version 1.8)

dojox/mobile/_ExecScriptMixin,

Mixin for a very simple content pane to embed an HTML fragment.

By mixing this class into a widget, the widget can have the ability to embed an external HTML fragment and to run the parser.

Usage:

var foo = new dojox/mobile/_ContentPaneMixin();
Defined by dojox/mobile/_ContentPaneMixin

Creates a new instance of the class.

See the dojox/mobile/_ContentPaneMixin reference documentation for more information.

Property Summary

  • content An HTML fragment to embed.
  • executeScripts If true, executes scripts that is found in the content.
  • href URL of the content to embed.
  • lazy If true, external content specified with the href property is not loaded at startup time.
  • parseOnLoad If true, runs the parser when the load completes.
  • prog If true, shows progress indicator while loading an HTML fragment specified by href.

Method Summary

  • _setContentAttr(data)
  • _setHrefAttr(href)
  • constructor() Creates a new instance of the class.
  • errorHandler(err) An error handler called when load fails.
  • execScript(html) Finds script tags and executes the script.
  • load() Loads external content specified with href.
  • loadHandler(response) A handler called when load completes.

Event Summary

  • onLoad() Stub method to allow the application to connect to the loading of external content (see load()).
