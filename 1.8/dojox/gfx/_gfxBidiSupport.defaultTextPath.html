dojox/gfx/_gfxBidiSupport.defaultTextPath (version 1.8)

Defines the default TextPath prototype.

See the dojox/gfx/_gfxBidiSupport.defaultTextPath reference documentation for more information.

Property Summary

  • align The horizontal text alignment, one of 'start', 'end', 'center'.
  • decoration The text decoration , one of 'none', ... .
  • kerning Whether kerning is used on the text, boolean default value true.
  • rotated Whether the text is rotated, boolean default value false.
  • text The text to be displayed, default value empty string.
  • type Specifies this is a TextPath, value 'textpath'.
