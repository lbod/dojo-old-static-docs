dojox/gfx/_gfxBidiSupport.defaultFont (version 1.8)

An object specifying the default properties for a Font used in text operations.

See the dojox/gfx/_gfxBidiSupport.defaultFont reference documentation for more information.

Property Summary

  • family The font family, one of 'serif', 'sanserif', ..., default value 'serif'.
  • size The font size (including units), default value '10pt'.
  • style The font style, one of 'normal', 'bold', default value 'normal'.
  • type Specifies this object is a Font, value 'font'.
  • variant The font variant, one of 'normal', ... , default value 'normal'.
  • weight The font weight, one of 'normal', ..., default value 'normal'.
