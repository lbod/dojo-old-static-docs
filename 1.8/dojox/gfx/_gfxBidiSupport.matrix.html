See the dojox/gfx/_gfxBidiSupport.matrix reference documentation for more information.

Property Summary

  • flipX a matrix, which reflects points at x = 0 line: flipX * (x, y) == (-x, y)
  • flipXY a matrix, which reflects points at the origin of coordinates: flipXY * (x, y) == (-x, -y)
  • flipY a matrix, which reflects points at y = 0 line: flipY * (x, y) == (x, -y)
  • identity an identity matrix constant: identity * (x, y) == (x, y)
