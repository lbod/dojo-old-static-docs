dojox/gfx/_gfxBidiSupport.Mover (version 1.8)

a constructor of custom Mover

Usage:

var foo = new dojox/gfx/_gfxBidiSupport.Mover(shape,e,host);
Defined by dojox/gfx/Mover

an object, which makes a shape follow the mouse, used as a default mover, and as a base class for custom movers

Parameter Type Description
shape dojox/gfx.Shape

a shape object to be moved
e Event

a mouse event, which started the move; only clientX and clientY properties are used
host Object
Optional

object which implements the functionality of the move, and defines proper events (onMoveStart and onMoveStop)

See the dojox/gfx/_gfxBidiSupport.Mover reference documentation for more information.

Method Summary

  • constructor(shape, e, host) an object, which makes a shape follow the mouse, used as a default mover, and as a base class for custom movers
  • destroy() stops the move, deletes all references, so the object can be garbage-collected

Event Summary

  • onFirstMove() it is meant to be called only once
  • onMouseMove(e) event processor for onmousemove
