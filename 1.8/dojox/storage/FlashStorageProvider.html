dojox/storage/FlashStorageProvider (version 1.8)

dojox/storage/Provider,

Storage provider that uses features in Flash to achieve permanent storage

Authors of this storage provider- Brad Neuberg, bkn3@columbia.edu

Usage:

var foo = new dojox/storage/FlashStorageProvider();
Defined by dojox/storage/Provider

See the dojox/storage/FlashStorageProvider reference documentation for more information.

onHideSettingsUI
Defined by: dojox/storage/Provider

If a function is assigned to this property, then when the settings provider's UI is closed this function is called. Useful, for example, if the user has just cleared out all storage for this provider using the settings UI, and you want to update your UI.
