Property Summary

  • baseLayerName The name of the base layer.
  • baseLayerOptions Additional specific options passed to OpensLayers layer, such as The list of layer to display, for Web Map Server layer.
  • baseLayerType type of the base layer.
  • baseLayerUrl Some layer may need an url such as Web Map Server.
