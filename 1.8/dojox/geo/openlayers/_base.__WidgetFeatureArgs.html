dojox/geo/openlayers/_base.__WidgetFeatureArgs (version 1.8)

The keyword arguments that can be passed in a WidgetFeature constructor. You must define a least one widget retrieval parameter and the geo-localization parameters.

See the dojox/geo/openlayers/_base.__WidgetFeatureArgs reference documentation for more information.

Property Summary

  • createWidget Function for widget creation.
  • dijitId The digitId of an existing widget.
  • dojoType The class of a widget to create.
  • height The height of the widget.
  • latitude The latitude, in decimal degrees where to place the widget.
  • longitude The longitude, in decimal degrees where to place the widget.
  • widget An already created widget.
  • width The width of the widget.
