dojox/geo/openlayers/TouchInteractionSupport (version 1.8)

class to handle touch interactions on a OpenLayers.Map widget

Usage:

var foo = new dojox/geo/openlayers/TouchInteractionSupport(map);
Defined by dojox/geo/openlayers/TouchInteractionSupport

Constructs a new TouchInteractionSupport instance

Parameter Type Description
