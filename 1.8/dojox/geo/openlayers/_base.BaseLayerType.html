dojox/geo/openlayers/_base.BaseLayerType (version 1.8)

Defines the base layer types to be used at Map construction time or with the setBaseLayerType function.

This object defines the base layer types to be used at Map construction time or with the setBaseLayerType function.

See the dojox/geo/openlayers/_base.BaseLayerType reference documentation for more information.

Property Summary

  • ARCGIS The ESRI ARCGis base layer selector.
  • BING Same as Virtual Earth
  • GOOGLE The Google base layer type selector.
  • OSM Base layer type as defined in dojox.geo.openlayer.BaseLayerType.
  • VIRTUAL_EARTH The Virtual Earth base layer type selector.
  • WMS The Web Map Server base layer type selector.
  • YAHOO The Yahoo base layer type selector.
