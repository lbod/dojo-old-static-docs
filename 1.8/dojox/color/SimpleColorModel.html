dojox/color/SimpleColorModel (version 1.8)

Base class for color models that return a color from a data value using an interpolation between two extremum colors.

Usage:

var foo = new dojox/color/SimpleColorModel(startColor,endColor);
Defined by dojox/color/SimpleColorModel

Construct a color model interpolating between start and end color. If only start color is provided use it to compute reasonable start and end colors from it.

Parameter Type Description
startColor dojo/_base/Color

The start color.

endColor dojo/_base/Color
Optional

The end color.

See the dojox/color/SimpleColorModel reference documentation for more information.
