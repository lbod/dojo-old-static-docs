dojox/atom/io/model._actions (version 1.8)

Container for tag handling functions.

Container for tag handling functions. Each child of this container is a handler function for the given type of node. Each accepts two parameters:

Property Summary

  • node The dom node containing the data
  • obj The object to insert data into.
