dojox/grid/enhanced/plugins/filter/_ConditionExpr._OperatorExpr (version 1.8)

The most abstract class for all operator expressions. An operator expression is a _ConditionExpr that represents an operation.

Usage:

var foo = new dojox/grid/enhanced/plugins/filter/_ConditionExpr._OperatorExpr();
Defined by dojox/grid/enhanced/plugins/filter/_ConditionExpr

The arguments are operands (or an array of operands, if the first argument is an Array) of this operator, ordering from left to right. Every operand should be a _ConditionExpr.
