dojox/grid/enhanced/plugins/filter/_ConditionExpr._BiOpExpr (version 1.8)

The most abstract class for all bi-operator expressions. A bi-operator expression is an _OperatorExpr that allow and only allow two operands.

Usage:

var foo = new dojox/grid/enhanced/plugins/filter/_ConditionExpr._BiOpExpr();
Defined by dojox/grid/enhanced/plugins/filter/_ConditionExpr

The arguments are operands (or an array of operands, if the first argument is an Array) of this operator, ordering from left to right. Every operand should be a _ConditionExpr.
