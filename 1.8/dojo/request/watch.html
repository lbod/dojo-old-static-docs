dojo/request/watch (version 1.8)

Watches the io request represented by dfd to see if it completes.

Parameter Type Description
dfd Deferred

The Deferred object to watch.

See the dojo/request/watch reference documentation for more information.

Property Summary

  • ioCheck Function used to check if basic IO call worked.
  • resHandle Function used to process response.
  • validCheck Function used to check if the IO request is still valid.

Method Summary

  • cancelAll() Cancels all pending IO requests, regardless of IO type
  • constructor(dfd) Watches the io request represented by dfd to see if it completes.
