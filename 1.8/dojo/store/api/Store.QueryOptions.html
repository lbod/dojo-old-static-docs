dojo/store/api/Store.QueryOptions (version 1.8)

Optional object with additional parameters for query results.

Property Summary

  • count The number of how many results should be returned.
  • sort A list of attributes to sort on, as well as direction For example: [{attribute:"price, descending: true}].
  • start The first result to begin iteration on
