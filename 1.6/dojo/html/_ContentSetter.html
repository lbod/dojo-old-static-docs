dojo.html._ContentSetter (version 1.6)

Object » dojo.html._ContentSetter
dojo.require("dojo.html");
Defined in dojo/html.js

Usage:

var foo = new dojo.html._ContentSetter(/* Object */ params, /* String|DomNode */ node);

Provides a configurable, extensible object to wrap the setting on content on a node call the set() method to actually set the content..

Property Summary

  • dojo.html._ContentSetter.parserScope Flag passed to parser. Root for attribute names to search for. If scopeName is dojo, will search for data-dojo-type (or dojoType). For backwards compatibility reasons defaults to dojo._scopeName (which is "dojo" except when multi-version support is used, when it will be something like dojo16, dojo20, etc.)
  • cleanContent Should the content be treated as a full html document, and the real content stripped of <html>, <body> wrapper before injection
  • content The content to be placed in the node. Can be an HTML string, a node reference, or a enumerable list of nodes
  • extractContent Should the content be treated as a full html document, and the real content stripped of <html>, <body> wrapper before injection
  • id Usually only used internally, and auto-generated with each instance
  • node An node which will be the parent element that we set content into
  • parseContent Should the node by passed to the parser after the new content is set
  • parseResults
  • startup Start the child widgets after parsing them. Only obeyed if parseContent is true.

Method Summary

  • _mixin(params)
  • _parse() runs the dojo parser over the node contents, storing any results in this.parseResults Any errors resulting from parsing are passed to _onError for handling
  • constructor(params, node) returns dojo.html._ContentSetter Provides a configurable, extensible object to wrap the setting on content on a node call the set() method to actually set the content..
  • empty()
  • set(cont, params) front-end to the set-content sequence
  • setContent() sets the content on the node
  • tearDown()

Event Summary
