Usage:

var foo = new dojo.dnd.__SourceArgs();

a dict of parameters for DnD Source configuration. Note that any property on Source elements may be configured, but this is the short-list

Property Summary

  • accept list of accepted types (text strings) for a target; defaults to ["text"]
  • autoSync if true refreshes the node list on every operation; false by default
  • copyOnly copy items, if true, use a state of Ctrl key otherwise, see selfCopy and selfAccept for more details
  • delay the move delay in pixels before detecting a drag; 0 by default
  • generateText generate text node for drag and drop, true by default
  • horizontal a horizontal container, if true, vertical otherwise or when omitted
  • isSource can be used as a DnD source. Defaults to true.
  • selfAccept accept its own items when copyOnly is true, true by default, works only if copyOnly is true
  • selfCopy copy items by default when dropping on itself, false by default, works only if copyOnly is true
  • withHandles allows dragging only by handles, false by default
