dojo.store.JsonRest (version 1.6)

Object » dojo.store.JsonRest
dojo.require("dojo.store.JsonRest");
Defined in dojo/store/JsonRest.js

Usage:

var foo = new dojo.store.JsonRest(/* dojo.store.JsonRest */ options);

This is a basic store for RESTful communicating with a server through JSON formatted data.

Property Summary

  • idProperty Indicates the property to use as the identity property. The values of this property should be unique.
  • target The target base URL to use for all requests to the server. This string will be prepended to the id to generate the URL (relative or absolute) for requests sent to the server

Method Summary

  • add(object, options) Adds an object. This will trigger a PUT request to the server if the object has an id, otherwise it will trigger a POST request.
  • constructor(options) returns dojo.store.JsonRest This is a basic store for RESTful communicating with a server through JSON formatted data.
  • get(id, options) Retrieves an object by its identity. This will trigger a GET request to the server using the url this.target + id.
  • getIdentity(object) Returns an object's identity
  • put(object, options) Stores an object. This will trigger a PUT request to the server if the object has an id, otherwise it will trigger a POST request.
  • query(query, options) Queries the store for objects. This will trigger a GET request to the server, with the query added as a query string.
  • remove(id) Deletes an object by its identity. This will trigger a DELETE request to the server.
