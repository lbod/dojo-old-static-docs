Method Summary

  • dojox.math.stats.approxLin(a, pos) returns Number Returns a linearly approximated value from an array using a normalized float position value.
  • dojox.math.stats.bestFit(a, xProp, yProp) returns Object Calculate the slope and intercept in a linear fashion. An array of objects is expected; optionally you can pass in the property names for "x" and "y", else x/y is used as the default. If you pass an array of numbers, it will be mapped to a set of {x,y} objects where x = the array index.
  • dojox.math.stats.forecast(a, x, xProp, yProp) returns Number Using the bestFit algorithm above, find y for the given x.
  • dojox.math.stats.max(a) returns Number Returns the max value in the passed array.
  • dojox.math.stats.mean(a) returns Number Returns the mean value in the passed array.
  • dojox.math.stats.median(a) returns Number Returns the value closest to the middle from a sorted version of the passed array.
  • dojox.math.stats.min(a) returns Number Returns the min value in the passed array.
  • dojox.math.stats.mode(a) returns Number Returns the mode from the passed array (number that appears the most often). This is not the most efficient method, since it requires a double scan, but is ensures accuracy.
  • dojox.math.stats.sd(a) returns Number Returns the standard deviation of the passed arguments.
  • dojox.math.stats.sum(a) returns Number Return the sum of all the numbers in the passed array. Does not check to make sure values within a are NaN (should simply return NaN).
  • dojox.math.stats.summary(a, alreadySorted) returns sort it properly|Object Returns a non-parametric collection of summary statistics: the classic five-number summary extended to the Bowley's seven-figure summary.
  • dojox.math.stats.variance(a) returns Number Find the variance in the passed array of numbers.
