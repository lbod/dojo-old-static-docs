Usage:

var foo = new dojox.uuid.Uuid(/* String? */ input);

This is the constructor for the Uuid class. The Uuid class offers methods for inspecting existing UUIDs.

Method Summary

  • dojox.uuid.Uuid.compare(uuidOne, uuidTwo, otherUuid) returns integer|integer (either 0, 1, or -1) Compares this UUID to another UUID, and returns 0, 1, or -1.
  • dojox.uuid.Uuid.getGenerator() returns generator (A UUID generator, such as dojox.uuid.TimeBasedGenerator). Returns the default generator. See setGenerator().
  • dojox.uuid.Uuid.getVariant() Returns a variant code that indicates what type of UUID this is. Returns one of the enumerated dojox.uuid.variant values.
  • dojox.uuid.Uuid.isEqual(otherUuid) returns boolean Returns true if this UUID is equal to the otherUuid, or false otherwise.
  • dojox.uuid.Uuid.isValid() Returns true if the UUID was initialized with a valid value.
  • dojox.uuid.Uuid.setGenerator(generator) Sets the default generator, which will be used by the "new dojox.uuid.Uuid()" constructor if no parameters are passed in.
  • constructor(input) returns dojox.uuid.Uuid A 36-character string that conforms to the UUID spec. examples: var uuid; uuid = new dojox.uuid.Uuid("3b12f1df-5232-4804-897e-917bf397618a"); uuid = new dojox.uuid.Uuid(); // "00000000-0000-0000-0000-000000000000" uuid = new dojox.uuid.Uuid(dojox.uuid.generateRandomUuid()); uuid = new dojox.uuid.Uuid(dojox.uuid.generateTimeBasedUuid()); dojox.uuid.Uuid.setGenerator(dojox.uuid.generateRandomUuid); uuid = new dojox.uuid.Uuid(); dojox.uuid.assert(!uuid.isEqual(dojox.uuid.NIL_UUID));
