The keyword arguments object used to create a palette based on the split complementary rules as implemented at http://kuler.adobe.com.

Property Summary

  • base The base color to be used to generate the palette.
  • da The delta angle to be used to determine where the split for the complementary rules happen. In degrees, the default is 30.
