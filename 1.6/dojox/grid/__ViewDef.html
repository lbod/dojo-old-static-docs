Usage:

var foo = new dojox.grid.__ViewDef();

Property Summary

  • cells The structure of the cells within this grid.
  • defaultCell A cell definition with default values for all cells in this view. If a property is defined in a cell definition in the "cells" array and this property, the cell definition's property will override this property's property.
  • noscroll If true, no scrollbars will be rendered without scrollbars.
  • type A string containing the constructor of a subclass of dojox.grid._View. If this is not specified, dojox.grid._View is used.
  • width A CSS size. If it's an Integer, the width will be in em's. If "noscroll" is true, this value is ignored.
