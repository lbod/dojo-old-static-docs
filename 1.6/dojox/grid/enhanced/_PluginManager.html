dojox.grid.enhanced._PluginManager (version 1.6)

Object » dojox.grid.enhanced._PluginManager
dojo.require("dojox.grid.enhanced._PluginManager");
Defined in dojox/grid/enhanced/_PluginManager.js

Usage:

var foo = new dojox.grid.enhanced._PluginManager(/* */ inGrid);

Plugin manager is responsible for 1. Loading required plugins 2. Handling collaboration and dependencies among plugins Some plugin dependencies: - "columnReordering" attribute won't work when either DnD or Indirect Selections plugin is on.
