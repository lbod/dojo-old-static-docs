Property Summary

  • dojox.grid.enhanced.plugins.exporter._ExportWriter.argObj.cell Reference to the current cell.
  • dojox.grid.enhanced.plugins.exporter._ExportWriter.argObj.cellIdx The index of the current cell in the current subrow. It's different from cell.index, which is the index in the whole line.
  • dojox.grid.enhanced.plugins.exporter._ExportWriter.argObj.colOffset If the grid has a _RowSelector view or something else, this view will NOT be passed to the user in argObj. So the column index (cell.index) will appear shifted (start from 1 instead of 0). This colOffset is provided to remove this shift. usage: var correctColIndex = argObj.cell.index + argObj.colOffset;
  • dojox.grid.enhanced.plugins.exporter._ExportWriter.argObj.grid The grid object we are now handling.
  • dojox.grid.enhanced.plugins.exporter._ExportWriter.argObj.isHeader Indicating which context we're handling, header or content.
  • dojox.grid.enhanced.plugins.exporter._ExportWriter.argObj.row The current row of data (logically), a.k.a.: current item.
  • dojox.grid.enhanced.plugins.exporter._ExportWriter.argObj.rowIdx The index of the current row (item).
  • dojox.grid.enhanced.plugins.exporter._ExportWriter.argObj.spCols An array of special column indexes(flat,not regarding structure). Special columns are typically attached to grid as a kind of UI facility by the grid widget, instead of some real data. For example, indirect selectors and row indexers. Users can choose to export it or not.
  • dojox.grid.enhanced.plugins.exporter._ExportWriter.argObj.subrow Reference to the current subrow. A subrow describe the innter structure of a row in a view, it's an array of cells
  • dojox.grid.enhanced.plugins.exporter._ExportWriter.argObj.subrowIdx The index of the current subrow in the subrow array: _View.structure.cells.
  • dojox.grid.enhanced.plugins.exporter._ExportWriter.argObj.view Reference to the current _View object.
  • dojox.grid.enhanced.plugins.exporter._ExportWriter.argObj.viewIdx The index of the current _View object in the views array. If the grid does not have any rowselector view, it conforms to the index in the _ViewManager.views.
