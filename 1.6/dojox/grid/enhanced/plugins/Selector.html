Usage:

var foo = new dojox.grid.enhanced.plugins.Selector(/* */ grid, /* */ args);

Provides standard extended selection for grid. Supports mouse/keyboard selection, multi-selection, and de-selection. Acceptable plugin parameters: The whole plugin parameter object is a config object passed to the setupConfig function. Acceptable cell parameters defined in layout: 1. notselectable: boolean Whether this column is (and all the cells in it are) selectable.
