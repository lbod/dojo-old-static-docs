Usage:

var foo = new dojox.form.uploader.plugins.IFrame();

Only supported by IE, due to the specifc iFrame hack used. The dojox.form.uploader.plugins.HTML5 plugin should be used along with this to add HTML5 capabilities to browsers that support them. Progress events are not supported. Inherits all properties from dojox.form.Uploader and dojox.form.uploader.plugins.HTML5.
