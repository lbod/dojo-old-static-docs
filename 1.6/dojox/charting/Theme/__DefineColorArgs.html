The arguments object that can be passed to define colors for a theme.

Property Summary

  • base A base color to use if we are defining colors using dojox.color.Palette
  • colors A pre-defined set of colors; this is passed through to the Theme directly.
  • generator The generator function name from dojox.color.Palette.
  • high An optional value to determine the highest value used to generate a color (HSV model)
  • hue A hue to base the generated colors from (a number from 0 - 359).
  • low An optional value to determine the lowest value used to generate a color (HSV model)
  • num The number of colors to generate. Defaults to 5.
  • saturation If a hue is passed, this is used for the saturation value (0 - 100).

Method Summary

  • constructor(num, colors, hue, saturation, low, high, base, generator) returns dojox.charting.Theme.__DefineColorArgs The number of colors to generate. Defaults to 5.
