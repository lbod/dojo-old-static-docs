Usage:

var foo = new dojox.charting.plot2d.__BarCtorArgs();

Additional keyword arguments for bar charts.

Property Summary

  • animate Whether or not to animate the chart to place.
  • areas Whether or not to draw areas on this plot. Defaults to false.
  • fill Any fill to be used for elements on the plot (such as areas).
  • font A font definition to be used for labels and other text-based elements on the plot.
  • fontColor The color to be used for any text-based elements on the plot.
  • hAxis The horizontal axis name.
  • lines Whether or not to draw lines on this plot. Defaults to true.
  • markerFill An optional fill to use for any markers on the plot.
  • markerFont An optional font definition to use for any markers on the plot.
  • markerFontColor An optional color to use for any marker text on the plot.
  • markerOutline An optional outline to use for any markers on the plot.
  • markerShadow An optional shadow to use for any markers on the plot.
  • markerStroke An optional stroke to use for any markers on the plot.
  • markers Whether or not to draw markers at data points on this plot. Default is false.
  • maxBarSize The maximum size for a bar in pixels. Default is 1.
  • minBarSize The minimum size for a bar in pixels. Default is 1.
  • outline An optional stroke used to outline any series on the plot.
  • shadow An optional stroke to use to draw any shadows for a series on a plot.
  • stroke An optional stroke to use for any series on the plot.
  • tension Whether or not to apply 'tensioning' to the lines on this chart. Options include a number, "X", "x", or "S"; if a number is used, the simpler bezier curve calculations are used to draw the lines. If X, x or S is used, the more accurate smoothing algorithm is used.
  • vAxis The vertical axis name

Method Summary
