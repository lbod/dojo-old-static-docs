Defined in dojox/drawing/tools/custom/Vector.js

See Toolbar.js makeButtons function. The toolbar checks Vector.setup for a secondary tool and requires name, label, and funct. Currently it doesn't accept icon and only uses text from label for the button. Funct is the function that fires when the button is clicked. Setup and postSetup are optional and allow tool specific functions to be added to the Toolbar object as if they were written there.
