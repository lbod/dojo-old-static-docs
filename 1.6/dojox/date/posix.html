dojo.require("dojox.date.posix");

Method Summary

  • dojox.date.posix.getIsoWeekOfYear(dateObject) returns Integer Get the ISO8601 week number of the given date. The week containing January 4th is the first week of the year. See http://en.wikipedia.org/wiki/ISO_week_date
  • dojox.date.posix.getIsoWeeksInYear(dateObject) returns Integer Determine the number of ISO8601 weeks in the year of the given date. Most years have 52 but some have 53. See http://www.phys.uu.nl/~vgent/calendar/isocalendar_text3.htm
  • dojox.date.posix.getStartOfWeek(dateObject, firstDay) returns Date Return a date object representing the first day of the given date's week.
  • dojox.date.posix.setIsoWeekOfYear(dateObject, week) returns Date Set the ISO8601 week number of the given date. The week containing January 4th is the first week of the year.
  • dojox.date.posix.strftime(dateObject, format, locale) returns String Formats the date object using the specifications of the POSIX strftime function
