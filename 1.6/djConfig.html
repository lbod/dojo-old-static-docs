djConfig (version 1.6)

Object » djConfig

Property Summary

  • djConfig.preloadImages An optional array of urls to preload immediately upon page load. Uses dojox.image, and is unused if not present.
  • djConfig.urchin Used by dojox.analytics.Urchin as the default UA-123456-7 account number used when being created. Alternately, you can pass an acct:"" parameter to the constructor a la: new dojox.analytics.Urchin({ acct:"UA-123456-7" });
