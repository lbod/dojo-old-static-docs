dijit.WidgetSet (version 1.6)

Object » dijit.WidgetSet
dojo.require("dijit._base.manager");
Defined in dijit/_base/manager.js

Usage:

var foo = new dijit.WidgetSet();

A set of widgets indexed by id. A default instance of this class is available as dijit.registry

Examples:

Example 1

Create a small list of widgets: 
	var ws = new dijit.WidgetSet();
	ws.add(dijit.byId("one"));
 	ws.add(dijit.byId("two"));
	// destroy both:
	ws.forEach(function(w){ w.destroy(); });

Example 2

Using dijit.registry: 
	dijit.registry.forEach(function(w){ /* do something */ });
