dijit.tree._dndSelector (version 1.4)

Object » dijit.tree._dndContainer » dijit.tree._dndSelector
dojo.require("dijit.tree._dndSelector");
Defined in dijit/tree/_dndSelector.js

Usage:

var foo = new dijit.tree._dndSelector(/* */ tree, /* */ params);

This is a base class for dijit.tree.dndSource , and isn't meant to be used directly. It's based on dojo.dnd.Selector.

Property Summary

  • anchor
  • containerState
  • current The currently hovered TreeNode.rowNode (which is the DOM node associated w/a given node in the tree, excluding it's descendants)
  • currentWidget
  • events
  • map
  • node
  • parent
  • selection DomNode> (id, DomNode) map for every TreeNode that's currently selected. The DOMNode is the TreeNode.rowNode.
  • simpleSelection
  • singular Allows selection of only one element, if true. Tree hasn't been tested in singular=true mode, unclear if it works.
  • tree Node or node's id to build the container on params: dijit.tree.__SourceArgs A dict of parameters, which gets mixed into the object

Method Summary

  • _addItemClass(node, type) Adds a class with prefix "dojoDndItem"
  • _changeState(type, newState) Changes a named state to new state value
  • _removeAnchor() returns self Removes the Anchor CSS class from a node. According to dojo.dnd.Selector, anchor means that "an item is selected, and is an anchor for a 'shift' selection". It's not relevant for Tree at this point, since we don't support multiple selection.
  • _removeItemClass(node, type) Removes a class with prefix "dojoDndItem"
  • _removeSelection() returns self Unselects all items
  • constructor(tree, params) returns dijit.tree._dndSelector Initialization
  • destroy() Prepares the object to be garbage-collected
  • forInSelectedItems(f, o) Iterates over selected items; see dojo.dnd.Container.forInItems() for details
  • getItem(key) returns dojo.dnd.Item Returns the dojo.dnd.Item (representing a dragged node) by it's key (id). Called by dojo.dnd.Source.checkAcceptance().
  • getSelectedNodes() Returns the set of selected nodes. Used by dndSource on the start of a drag.
  • selectNone() returns self Unselects all items

Event Summary

  • onMouseDown(e) returns ignore right-click Event processor for onmousedown
  • onMouseMove(e) mouse event
  • onMouseOut(widget, evt) Called when mouse is moved away from a TreeNode
  • onMouseOver(widget, evt) Called when mouse is moved over a TreeNode
  • onMouseUp(e) Event processor for onmouseup
  • onOutEvent() This function is called once, when mouse is out of our container
  • onOverEvent() This function is called once, when mouse is over our container
