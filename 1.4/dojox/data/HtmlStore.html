Usage:

var foo = new dojox.data.HtmlStore(/* Object */ args);

The HtmlStore can be created in one of two ways: a) by parsing an existing table or list DOM node on the current page or b) by referencing an external url and giving the id of the table or list in that page. The remote url will be parsed as an html page. The HTML table or list should be of the following form:
Attribute1 Attribute2
Value1.1 Value1.2
Value2.1 Value2.2

_fetchItems
Defined by dojox.data.HtmlStore

If '_fetchUrl' is specified, it is used to load an XML document with a query string. Otherwise and if 'url' is specified, the XML document is loaded and list XML elements that match to a query (set of element names and their text attribute values that the items to contain).

A wildcard, "*" can be used to query values to match all occurrences. If '_rootItem' is specified, it is used to fetch items.
