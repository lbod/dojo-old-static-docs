Usage:

var foo = new dojox.grid.enhanced.dnd._DndGrid(/* */ dndManager);

This class declaration is used to be mixed in to dojox.grid._Grid to enable DND feature in _Grid.

Property Summary

  • dndSelectable whether the DND feature is enabled for the grid
  • select handle the DND selecting operation

Method Summary

  • constructor(dndManager) returns dojox.grid.enhanced.dnd._DndGrid constructor, store the reference of the instance of_DndSelectingManager
  • domousedown(e) handle the mouse down event
  • domouseup(e) handle the mouse up event

Event Summary

  • onCellMouseUp(e) Event fired when mouse is up in a cell.
  • onMouseUp(e) Event fired when mouse is up inside grid.
  • onMouseUpRow(e) Event fired when mouse is up inside grid row
  • onRowHeaderMouseDown(e) returns alwarys return for oncontextmenu event to only show menu Handle when there is a mouse down event on row header (row bar), Call the dnd select to add the row to select
  • onRowHeaderMouseUp(e) Handle when there is a mouse up event on row header (row bar),
  • onRowMouseUp(e) Event fired when mouse is up in a row.
