Usage:

var foo = new dojox.grid.enhanced._Plugin(/* */ inGrid);

Plugin manager is responsible for 1. Loading required plugins 2. Handling collaboration and dependencies among plugins 3. Overwriting some default behavior of DataGrid Note: Mixin and method caching are used for #3, this might be refined once an overall plugin architecture is set up for DataGrid. Some plugin dependencies: - DnD plugin depends on NestedSorting plugin - RowSelector should be used for DnD plugin. e.g.

- "columnReordering" attribute won't work when either DnD or Indirect Selections plugin is on.
